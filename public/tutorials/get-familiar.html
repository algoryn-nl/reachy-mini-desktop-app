Get familiar with the robot

Use the daemon API via console commands

The Reachy Mini daemon runs a REST API at localhost:8000. It manages robot communication and exposes endpoints to control motors, read sensors, and execute movements. Use curl commands or any HTTP client to interact with it.

Example Commands

Get robot state:
curl http://localhost:8000/api/state/full
Enable motors:
curl -X POST http://localhost:8000/api/motors/enable
Move head:
curl -X POST http://localhost:8000/api/goto \
  -H "Content-Type: application/json" \
  -d '{"head": {"x": 0, "y": -10, "z": 0}}'
