Share your app on Hugging Face Spaces

After building your app, deploy it on Hugging Face Spaces to share it with the community. This tutorial will guide you through the deployment process step by step.

Step 1: Set Up Hugging Face Space

Create a new Space on your Hugging Face account. Choose the Static SDK framework with a blank template.

Navigate to huggingface.co/new-space
Pick "Static SDK" from the framework options

Step 2: Upload Your Code

Set up git in your app folder, connect it to your Space, and upload all your files.

git init
git remote add space git@hf.co:spaces/your-username/your-app-name
git add .
git commit -m "first commit!"
git push -f space main:main

Step 3: Publish & Discover

After testing, switch your Space to public and add the reachy_mini tag so others can find it.
